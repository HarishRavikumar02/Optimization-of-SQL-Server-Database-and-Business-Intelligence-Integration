# Optimization of SQL Server Database and Business Intelligence Integration

This repository presents a database optimization and business intelligence project focused on improving the performance, scalability, and analytical capabilities of an online sales platform for MumsNet.

The project emphasizes relational database design best practices and the integration of business intelligence solutions to support operational and strategic decision-making.

---

## Project Overview

The objective of this project is to optimize an existing SQL Server database and build a business intelligence platform for analyzing online sales data.

The work focuses on:

- improving database structure through normalization
- enhancing query performance and scalability
- enabling structured reporting and analytics for business users

---

## Key Features

- SQL Server database optimization through normalization
- Improved database design for better performance and maintainability
- Business intelligence layer for online sales analytics
- Structured reporting to support operational monitoring and decision-making
- Scalable data model to support future data growth

---

## Technical Approach

- Analyzed existing database schema and identified design and performance issues
- Applied normalization techniques to reduce redundancy and improve data integrity
- Redesigned tables and relationships to support scalable reporting
- Integrated the optimized database with a BI platform for sales analysis and reporting

---

## Business Intelligence Integration

The BI solution enables:

- analysis of online sales performance
- monitoring of key operational metrics
- generation of reports for business stakeholders
- improved visibility into business operations

---

## Use Case

This solution supports:

- data-driven operational planning
- improved visibility into online sales trends
- faster and more reliable reporting for business teams
- scalable analytics for future growth
